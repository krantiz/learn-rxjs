Custom Observable.

Every Custom Obserable has access to 3 functions:

  1. Next(): It is similar to success and waits for next stream of data
  2. Error(): It is used to catch an exception, especially when any http request is subscribed
  3. Complete(): It is a way to stop the subscription when work is done.
